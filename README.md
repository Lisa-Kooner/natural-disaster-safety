# Natural Disaster Safety
- **Built for:** McMaster Engineering Competition, 2025

## Overview
**Natural Disaster Safety** is a Python-based GUI application designed to help users respond effectively during natural disasters. This app allows users to select the type of natural disaster they believe they are experiencing. After that, it provides them with tips on what to do depending on the scenario, including warning signs and suggestions on what to pack for an emergency kit.

This project focuses on making critical safety information accessible, intuitive, and easy to follow when time and clarity matter most.

---

## Features
- Disaster-specific safety guides with structured instructions  
- Step-by-step protective actions for emergency scenarios  
- User-friendly graphical interface for fast navigation  
- Designed to support quick decision-making under stress  

---

## Technologies Used
- Python  
- Python GUI framework (Tkinter)
